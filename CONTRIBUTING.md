# Contributing to Kenya EV Data

Thank you for your interest in contributing to Kenya's open EV data. This repository is maintained by [Electric Kenya](https://electric.ke) and we welcome community input.

## How to Contribute

### Reporting New Charging Stations

If you know of an EV charging station in Kenya that is not listed:

1. Open an [Issue](../../issues) with the title: `New Station: [Station Name]`
2. Include as much of the following as possible:
   - Station name and operator
   - County and specific location/address
   - Power output (kW)
   - Connector types (CCS2, CHAdeMO, Type 2, Battery Swap, etc.)
   - Number of connectors
   - Pricing (if known)
   - Operational status
   - A photo (optional but helpful)

### Correcting Existing Data

If any data in the CSV files is incorrect or outdated:

1. Open an [Issue](../../issues) describing the correction
2. Or submit a Pull Request with the corrected data

### Adding Vehicle Data

If a new EV model has become available in Kenya:

1. Open an [Issue](../../issues) with the title: `New Vehicle: [Make Model]`
2. Include: make, model, variant, type (Electric/Hybrid/PHEV), range (km), power (kW), and a source link

### Policy Updates

If there are changes to Kenya's EV-related tax rates, regulations, or standards, please open an issue with the relevant details and official source.

## Data Quality Standards

- All data should be verifiable from official or reputable sources
- Charging station data should be based on first-hand verification where possible
- Vehicle specifications should match manufacturer-stated figures
- Policy and regulatory data should reference official government publications

## Code of Conduct

Be respectful, constructive, and collaborative. We are building this for the benefit of Kenya's EV community.
